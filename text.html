Creating a Portfolio Website

In this collaboration, I created a portfolio website about myself. This experience was quite challenging because I worked alone, not in a group. Why was it challenging? Because during the first collaboration, we were in a group of three, including myself.

In the initial stage, I created the design in Figma. I struggled because it was my first time using it. However, over time, I adapted to Figma and learned the basics.

After completing the Figma design, I coded the HTML, CSS, and JS in Visual Studio Code. This was where my difficulty peaked, as it had to be identical to the design, which was very difficult for me. I sought solutions by asking friends or using AI.

After that, I began to understand a little about how to create this website. While doing this activity, I recorded it for an informatics assignment, which required creating a video showing the opening, the website creation process, and the closing. The video can be edited as creatively as possible.

From this experience, I learned about website creation, although I didn't do it entirely myself. I used AI to refine the website only to see how it was lacking. I also don't have too much of a headache because I made my own website.
